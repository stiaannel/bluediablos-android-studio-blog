This post will contain a list of beginner linux terminal commands

### # Print working directory

***

```sh
pwd
```

### # List files in current directory

***

```sh
ls
```

List hidden items (has a verbose output)

***

```sh
ls -a
```

### 1. Change Directory

***

```sh
cd <directory to change to>
```

### 2. Make Folder

***

```sh
mkdir <folder name>
```

### 3. Remove Folder

***

```sh
rmdir <folder to remove>
```

### 4. The Remove Command

***

```sh
rm <file to remove>
```

### 5. Create a New empty File

***

```sh
touch <file name here>
```

### 6. Help and Manuals

***

```sh
man <command>
```

alternatively

***

```sh
<command> --help
```

### 6. Copying a file

***

```sh
cp <file name> <location>
```

### 6. Moving a file

***

```sh
mv <file name> <location>
```

### 6. Renaming a file

***

```sh
mv <file name> <new name>
```

### 7. Searching for commands

***

```sh
apropos <what you want to do>
```

Some tips to remember

- You can use tab to finish of your commands.
- The command `apropos` is  handy if you get stuck.
